Как использовать

3 шага до запуска

📋
1. Выбери компонент
Используй поиск и фильтры. Нажми на карточку чтобы открыть GitHub-репозиторий с кодом и документацией.
⬇️
2. Склонируй и запусти
git clone ... → pip install -r requirements.txt

У каждого MCP есть README с инструкцией, примеры и setup-скриптами.
⚙️
3. Подключи к агенту
MCP работает через stdio. Добавь его в config.yaml своего Hermes Agent или другой платформы. Tools готовы к использованию.
